Prompt Generator V4 14-Layer System
🗺 Layer Architecture — 14 Layer System Visual Preview Only
L1A · PERMANEN 🔒
Identity Anchor
Paspor digital karakter. Race, age, face DNA, body DNA. Tidak boleh diubah setelah di-approve.
L1B · PERMANEN 🔒
Technical Constraints
Negative DNA — pagar identitas. Semua larangan keras agar AI tidak drift.
L0 · Desain Bebas
Subjects
Blueprint karakter — anatomi, kostum, pose, ekspresi. Bisa divariasi, locked field ikut L1A.
L2A · Variabel
Scene Core
Waktu, tempat, fokus narasi. Bisa diganti setiap generate.
L2B · Variabel
Environment
Detail lokasi, cuaca, props. Bebas divariasi.
L3 · Variabel
Composition
Framing kamera, focal length, DoF, shot type, placement.
L4A · VIDEO ONLY
Motion
Pergerakan kamera, gerakan subjek, cloth & hair physics.
L4B · VIDEO ONLY
Temporal Sequence
Durasi, FPS, loop, transisi, timing key events.
L5 · Variabel
Lighting
Preset, key/fill/rim light, warna, suhu, shadow, fog, god rays.
L6 · Variabel
Atmosphere
Emotional tone, visual style, color palette, grain, texture.
L7 · Platform
Technical
Resolusi, AR, ISO, shutter, color grading, post-processing.
L8C · Output Engine
Output Config
Section order, weights, negative banks, platform params. Merakit prompt final.
L8B · Internal
Variations
Batch config, seed, genre profiles, randomization rules. Weight 0.0.
L8A · Admin
Metadata
Title, author, tags, target platform. Weight 0.0.
